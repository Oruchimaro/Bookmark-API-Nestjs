# BOOKMARK APP

## Description

Test Project (Bokkmark App) Using Nestjs and Prisma.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

# Run Database

- Setup docker-compose.yml
- run docker compose up <image-name> -d  ==> docker compose up dev-db -d
